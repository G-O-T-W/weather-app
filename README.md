A small, clean Weather App built with HTML, CSS, and vanilla JavaScript using Webpack. It fetches data from the Visual Crossing Free Weather API and displays current conditions, hourly forecasts, weekly forecasts and a dynamically-changing background based on weather.

## Demo

- Live demo: https://g-o-t-w.github.io/weather-app/ üöÄ

## Features

- Displays current weather (temperature, humidity, aqi, sunrise and sunset).
- Shows an hourly forecast carousel.
- Dynamically changes the background image based on current weather conditions (Unsplash-sourced images).
- Sidebar showing weekly forecast
- Minimal, clean UI designed for readability and quick scanning.

## API & Assets

- Weather data: Visual Crossing Free Weather API
  - Query Builder: https://www.visualcrossing.com/weather-query-builder/ (explore available parameters and examples)
- Weather icons: Visual Crossing Colored Weather Icons Pack
  - Icon pack image: https://github.com/visualcrossing/WeatherIcons/blob/main/1st%20Set%20-%20Color%20-%20Index.png
- Visual Crossing: [visualcrossing.com](https://www.visualcrossing.com/) 

![Visual Crossing](https://img.shields.io/badge/Visual%20Crossing-API-blue?style=flat-square)

Background and wallpaper images are sourced from Unsplash.


## Local Development

Get the app running locally:

1. Clone the repository
2. Install dependencies and start the dev server

```bash
npm install
npm run dev
```

## Build

Create a production bundle:

```bash
npm run build
```
